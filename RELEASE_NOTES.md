# Lonely Wiki リリースノート テンプレート

## v0.1.0 - 2024年12月14日 🚀

### ✨ 新機能

#### コア機能
- **オフライン Wiki アプリケーション** - インターネット接続なしで完全に動作
- **Markdown エディタ** - GitHub Markdown 形式をサポート
- **リアルタイムプレビュー** - 入力中にプレビューをリアルタイム表示
- **タグベース検索** - 複数のタグで記事を効率的に分類・検索
- **キーワード検索** - タイトルと内容から記事を検索

#### UI/UX
- **2 列レイアウト** - サイドバーのタグパネルと記事一覧
- **ダークテーマ** - 目に優しいデフォルトテーマ
- **記事エディタ** - 3 セクション設計（ヘッダー、エディタ、メタデータ）
- **URL コピー** - ワンクリックで記事 URL をコピー

#### 検索・並び替え
- **複数基準でのソート** - 更新日時、作成日時、表示回数、タイトル
- **タグフィルター** - 記事をタグで絞り込み
- **キーワード検索** - 全文検索対応

### 🐛 バグ修正

- 初回リリースのためなし

### 🔧 インストール方法

#### Windows
1. `.exe` ファイルをダウンロード
2. ダブルクリックで実行
3. インストーラーの指示に従う

#### macOS
1. `.dmg` ファイルをダウンロード
2. ダブルクリックでマウント
3. Lonely Wiki アプリをアプリケーションフォルダにドラッグ

#### Linux (AppImage)
```bash
chmod +x Lonely\ Wiki-0.1.0.AppImage
./Lonely\ Wiki-0.1.0.AppImage
```

#### Linux (Debian)
```bash
sudo apt install ./lonely-wiki_0.1.0_amd64.deb
```

### 📋 システム要件

- **Windows**: Windows 7 以上
- **macOS**: macOS 10.13 以上
- **Linux**: glibc 2.28 以上のデスクトップ環境
